v 0.1

Our first version includes the most basic functionality for the @ Menu.

  • Basic index to view plugin
  • Basic functionality to show and hide a menu within the textarea
  • Settings Added
    1. settings.target ID, class, or query selector of the textarea
    2. settings.menu ID, class, or query selector of the menu
    3. settings.openMenu Key/ character to open the menu
    4. settings.closeMenu Key to close the menu
    5. settings.menuItem Class of the items within the menu - this allows for filtering, sorting, and navigation
    6. settings.marginTop Margin above the menu so that it's not over the line of text/ openMenu key
  • Basic Changelog page added
