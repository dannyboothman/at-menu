Options

Options for the At Menu JS Plugin.

Required Options

The following options are required for the At Menu to initialize.

Option
Code Example
Name
target
Type
string
Default
N/A
Summary
The element of the textarea you want the menu to appear on.
AtMenu.init({
    "target": document.getElementById("atm"),
    "menu": document.getElementById("atm-menu")
});
Option
Code Example
Name
menu
Type
string
Default
N/A
Summary
The div you want to appear.
AtMenu.init({
    "target": document.getElementById("atm"),
    "menu": document.getElementById("atm-menu")
});

Extra Options

The following options can be used to extend and customise the At Menu.

Option
Code Example
Name
openMenu
Type
string
Default
@
Summary
The character that when typed will display the menu.
AtMenu.init({
    "target": document.getElementById("atm"),
    "menu": document.getElementById("atm-menu"),
    "openMenu": "@",
});
Option
Code Example
Name
closeMenu
Type
string
Default
Escape
Summary
The character that when typed will close the menu.
AtMenu.init({
    "target": document.getElementById("atm"),
    "menu": document.getElementById("atm-menu"),
    "closeMenu": "Escape",
});
Option
Code Example
Name
marginTop
Type
int
Default
20
Summary
Margin Top of the menu. This allows the menu to not overflow the text.
AtMenu.init({
    "target": document.getElementById("atm"),
    "menu": document.getElementById("atm-menu"),
    "marginTop": "20",
});
