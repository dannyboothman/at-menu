Installation

Include CSS

First include the following CSS file into your projects HTML head

at-menu.css
<link type="text/css" rel="stylesheet" href="css/at-menu.css" />

Include JS

Include jQuery and at-menu.js

We'll be looking at removing jQuery dependancy in later versions
at-menu.js
<script src="js/jquery.js"></script>
<script src="js/atm.js"></script>

Setup HTML

You'll need to add a textarea and menu div

We'll be looking at allowing @ Menus on inputs and content editable regions in later versions
.html
<textarea id="atm" class="atm"></textarea>

<div id="atm-menu" class="atm-menu">
    <div class="atm-item">Menu Item 1</div>
    <div class="atm-item">Menu Item 2</div>
    <div class="atm-item">Menu Item 3</div>
    <div class="atm-item">Menu Item 4</div>
    <div class="atm-item">Menu Item 5</div>
</div>
